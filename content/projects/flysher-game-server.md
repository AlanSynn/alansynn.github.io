A game-server framework for TypeScript developers, providing the core development framework and many related components, including libraries and tools.

The former framework, [pomelo](https://github.com/NetEase/pomelo), had been deprecated, so we built a new framework based on the former architecture to achieve a fast and adaptable foundation.

### Shipped for a global top-10 social game

- Developed 4 social games and features.
- Built a data pipeline and analysis for a global top-10 social game.
- Designed and implemented unit- and simulation-testing procedures for business logic.
- Developed type-checking software to increase type safety (Node.js).
- Built a QA system with a virtual agent.
